Features

Simple, powerful generator.

Generate placeholders instantly with the following parameters:

  • Scene Number
  • First Shot
  • Increment
  • Shot Padding
  • Shot Count
  • Cut In Frame Number
  • Frame Rate
Renumber Clips.

If you need to renumber clips in a timeline Shotify lets you easily accomplish this. Simply select all the clips in the timeline, choose the new parameters and click rename.
Shotify will sequentially rename all the clips. Useful if a shot is added or omitted.

Thumbnails in a click

Export thumbnails from the first frame of every shot including labels, and named correctly. Simply choose an export location and click "Export Thumbnails".
Shotify will export pngs in an instant and rename to match you shotnumber, great for spreadsheets, ftrack, shotgrid you name it.

Export Movie refs.

Similarly Shotify lets you export shot references movie files for each clip with a click. Choose an export location, click "Export Shots" and let Shotify do it's thing.
